
# Pavan Kumar Portfolio

Minimal Astro portfolio for Full Stack AI Engineering positioning.

## Running

```sh
bun install
bun run dev
```

## Environment

```sh
PUBLIC_CONTACT_EMAIL=hello@example.com
PUBLIC_BLOG_RSS_URL=https://your-publication.substack.com/feed
PUBLIC_SUBSTACK_URL=https://your-publication.substack.com
```

Blog cards are fetched from the external RSS feed at build time. Clicking a post opens the original publication; full posts are not rendered locally.

## Build

```sh
bun run build
bun run preview
```
